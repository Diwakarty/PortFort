Mitigation Steps:

  • Port 25 (SMTP): Use encryption (e.g., STARTTLS) for email communication.
  • Port 53 (DNS): Secure DNS servers and implement rate limiting.
  • Port 80 (HTTP): Redirect to HTTPS for secure communication.
  • Port 23 (Telnet): Disable Telnet and use SSH for secure remote connections.
  • Port 3306 (MySQL): Secure with strong credentials and restrict access.
  • Ports 139 and 445 (SMB): Disable SMBv1 and restrict access to trusted hosts.
  • Port 22 (SSH): Use key-based authentication and disable root login.
  • Port 21 (FTP): Disable FTP or switch to SFTP for secure file transfers.
